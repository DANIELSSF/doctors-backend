  ## Description

  This project is a backend application for managing doctors, built using the NestJS framework. It provides APIs for creating, updating, and retrieving doctor information.

  ## Installation

  ```bash
  $ npm install
  ```

  ## Running the app

  ```bash
  # development
  $ npm run start

  # watch mode
  $ npm run start:dev

  # production mode
  $ npm run start:prod
  ```

  ## Running the database with Docker

  ```bash
  $ docker-compose up -d
  ```

  ## Test

  ```bash
  # unit tests
  $ npm run test
  ```
